# Shashank Sakilam - Portfolio

A modern, responsive portfolio website showcasing professional experience, skills, projects, and achievements with dark/light mode support.

## Features

- **Hero Section** - Eye-catching introduction with profile image and CTA buttons
- **Prologue** - About me section and education details
- **Arsenal** - Skills showcase with proficiency levels
- **Forge** - Featured projects with descriptions and technologies
- **Battlefield** - Professional experience timeline
- **Trophies** - Achievements and certifications
- **Epilogue** - Contact section with social links
- **Dark/Light Mode** - Theme toggle for user preference
- **Scroll Animations** - Smooth fade-in animations on scroll
- **Responsive Design** - Mobile-friendly layout

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
- **Vite** - Build tool

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Customization

- Update personal information in the Hero section
- Modify skills in the Arsenal section
- Add/edit projects in the Forge section
- Update experience in the Battlefield section
- Add achievements in the Trophies section
- Update contact links in the Epilogue section
